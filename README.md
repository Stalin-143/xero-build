<h3 align="center">January 2025</h3>

**Happy New Year 2025**. This month's version includes a few changes and some minor fixes. See list below.

#### Chages/Fixes

- Kernel 6.12.7.
- Added **Wine** 9.22-1.
- QT Updated to 6.8.1.
- Xero-Toolkit updated.
- Stability improvements.
- Frameworks Updated to 6.9.0.
